Configuration

To configure SBA, go to admin>Configuration>Stock, you should see the following. All options starting with SBA have been added to the stock menu by this mod.

By default all new flags are set to off (false). Here are some suggeted setting:

For information on how to use the following option switches click below:

  • Check Stock level
  • Subtract Stock
  • Allow Checkout
  • Mark product out of stock
  • Stock Re-Order level
  • SBA Show Available Stock Level in Cart (when less than order)
  • SBA Display Images in Admin
  • SBA Show Stock Level on Product Info Page
  • SBA Original Price Struck Through
  • SBA Display Search Box Only
  • SBA Display Search List Box
  • SBA Display Custom ID

 

Check Stock level - Check to see if sufficient stock is available . Options (True/False)

  • True - Checks to see if stock is available. If not it will display Out of Stock. It allows checkout, if stock is unavailable (backorders item).



  • False - Checks to see if stock is available. If not, it will NOT display Out of Stock. It allows checkout, if stock is unavailable (backorders item).



    Go To Index

Subtract Stock - Subtracts from products quantity in stock the number of products ordered. Options (True/False)

  • True - If an item is sold, this option will deduct from the total inventory stock for that item.
  • False - Will not deduct from inventory stock.

    Go To Index

Allow Checkout - Allow customer to checkout even if there is insufficient stock Options (True/False)

  • True - Allows checkout, even if the item is not in stock. This option checks out the item and places the item on backorder. It also notifies the customer that the Item will be backordered, with the following warning in the shopping cart.





  • False - Does not allow checkout if no inventory is available. It grays out the stock attribute, which are out of in stock, making it not selectable.



    In the case, if customer orders more that what's available in stock, the cart will not allow checkout, and will ask the customer to change the quantity for that item. It notifies the client with the following warning in the cart.

Mark product out of stock - Display a sign on screen, so customer can see which product has insufficient stock.

Stock Re-Order level - Defines when stock needs to be re-ordered.

By default it is set to 5 items.

Go To Index

SBA Show Available Stock Level in Cart (when less than order) - When customer places more items in cart than are available, show the available stock on the shopping cart page. Options (True/False)

  • True - Displays the Quantity available per item in the shopping cart.


  • False - Does not Display the Quantity available per item in the shopping cart.

    Go To Index

SBA Display Images in Admin - Displays the product image, under admin>Catalog>Products with Attributes Stock. Options (True/False).



Go To Index

 

SBA Show Stock Level on Product Info Page - Show the available stock for each attribute on product info page. Options (True/False)

  • True - Displays the Quantity available per item
  • False - Does not Display the Quantity available per item



    Go To Index

 

SBA Original Price Struck Through - Shows the original price (struck through) on product info page with attribute. Options (True/False)

Go To Index

SBA Display Search Box Only - Show Search box only (no records). Options (True/False)

Under admin>Catalog>Products With Attributes Stock,
  • True - Only shows all the search box(s), without displaying any of the records of products with attributes stock variants.


  • False- Shows all the search box(s) and all records for products and their corresponding attributes stock variants.



    Go To Index

SBA Display Search List Box - Show the Search List box At the top of the page. Options (True/False)

  • True - Displays a drop down list of all your existing products with stock attribute to choose from.
  • False - Does NOT display the drop down list.



Go To Index

SBA Display Custom ID - Display Custom Id (if defined) in cart, checkout processand invoice. Options (True/False)
