Attribute Stock Setup

Before adding any attribute stock variant to SBA, products and their corresponding attributes must be defined and setup in Zen Cart.

Setting up products

As an example we will setup for a Picture Frame.

1) Setup Category and Product:

  • Create a Product category called (Picture Frames)
  • Create a Product called (Picture Frame Style 1)
    Note: while defining your product, be sure to set Product Priced by Attributes: to Yes. And NOT to place any quantity for Products Quantity: in this area. As SBA is handling that part.

2) Setup your attributes for (Picture Frame Style 1).

For this example, attributes Option Name, and Option Type are setup as follow:

- Option Name: Size - Option Type: Radio
- Option Name: Color - Option Type: Read Only
- Option Name: Material - Option Type: Read Only

 

  • Setup the above and create Sizes 4x6, 5x7, 6x9 and 8x10.
  • Set the Default Attribute (In this case Size 4x6, shown in Orange)
  • Set to Display Only, all Attributes that have Attribute Option Type of Read Only. Shown in Yellow!
    VERY IMPORTANT:     If you miss this step, the Out of Stock message will get displayed on your product info page; for Color and Material attributes.




3) Go to admin>Catalog>Products With Attributes Stock. Depending on product(s) user have setup in the cart, user would see a list similar to the following:

At first, since there are no inventory is set for product attributes variants, it shows Quantity for all variants as 0. Except for products that do NOT carry stock by attribute.

Once user start adding stock for your product attribute variants, it will add up all quantities and will display the total in Quantity for all variants column.


4) To add product attribute variant stock

  • Click on Add Quantity For Product Variant link


  • User will see a window similar to the following (depending on the number of product attribute varients available) . At this point user should only add the variant controlling the quantity. If an attribute is not controlling the stock, leave it as N/A. Place any custom ID for that product and it's stick quantity and click submit.



    As an example, if user sets the following quantities for the Size variant to
    Size 4x6 - 1
    Size 5x7 - 0
    Size 4x6 - 2
    Total for all variants of Picture Frame Style 1 would be 3, as follow.



    Note:
    Jquery option may be used for updating the:
    "Quantity In Stock", "Sort Order", or "Custom ID"
    Example: On the SBA entry page, Click on the "Quantity In Stock", the field should become editable.
    Make changes, than click the "Save" button on the page.

    The product info page will look like:



    If noticed, we did not add the stock for variant 6x9, and it shows it as Out of Stock!


5) Sync Quantities

Sync Quantities adds up all variant quantities for a product and places the result in Quantity for all variants. This information gets displayed in the product information page and admin>Catalog>Products With Attributes Stock.

This function could be triggered for individual products or for all product(s) quantities.
